This is a cloud service that processes your MFPO calendar subscription for better formatting.

Usage:

https://mfpo.vercel.app/api/?filter=<regex>&link=<MFPO link>

filter: A regex that matches the course code of an event, useful when you only want to receive calendar events for a specific course.

link: The calendar subscription link you obtained in MFPO.

Use the link above instead of the one obtained in MFPO when subscribing to the calendar. Works across all calendar apps that support iCal subscription.
